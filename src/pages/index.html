The Auraria Library website at http://library.auraria.edu, directing 2 millions page views traffic per year, is the main information portal that leads to a wealth of resources and services provided by the Library.

The current version of the library website, built with Content Management System (CMS) Drupal 6, is 6 years old. In Feb 2016, Drupal 6 has also reached its end of life, which means, the library website needs to be migrated away from this outdated technology with the highest priority. Because the needs of library users as well as the underlying technology of CMS have all evolved greatly within the last 6 years, it is not possible to just migrate content from the old system into the new. The prudent path forward would be to utilize the latest technology to completely redesign and rebuild the main library website from the ground up.

Key Objectives

Content-Driven Design

Incorporate content strategy into the entire lifecycle of the design process to ensure the delivery of an engaging website experience.

User Impact Maximization

Prioritize features development scopes based on how they will impact the daily experience of various users to increase ROI while accelerating TTM (Time-To-Market).

Responsive Design Adoption

Deploy the latest responsive design technology and best practises to deliver coherent cross-device and cross-platform user experience.

Infrastructure Improvement

Leverage the benefits of an enterprise-level Drupal hosting solution to maximize system reliability while increasing team productivity.
