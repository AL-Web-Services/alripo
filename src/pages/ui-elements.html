What is a living styleguide?

“A [living] style guide is a living document of code, which details all the various elements and coded modules of your site or application. Beyond its use in consolidating the front-end code, it also documents the visual language, such as header styles and color palettes, used to create the site.” ~ A List Apart, Issue №393
